Open Data Cube: Introdução e Integração com o projeto BDC
=========================================================


.. image:: ./img/odc/logo-odc-bdc.png
    :alt: Open Data Cube
    :height: 128
    :align: right


**Descrição:** Atualmente o *framework* **O**\ pen **D**\ ata **C**\ ube (ODC) e todo seu ecossistema de ferramentas vem sendo testado com os produtos de dados do Brazil Data Cube. O objetivo desta integração é aumentar o conjunto de ferramentas de acesso, visualização e análise disponíveis para os usuários dos diversos cubos de dados produzidos pelo projeto. Este *Tech Talk* irá apresentar os principais conceitos relativos ao ODC e o status da integração desta tecnologia no projeto BDC. Os participantes serão convidados a realizar um *hands-on* durante a apresentação.


**Data:** 04 de Setembro de 2020


**Apresentadores:** Felipe Menino Carlos e Vitor Gomes
