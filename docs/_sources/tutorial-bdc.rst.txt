Tutorial das ferramentas BDC para desenvolvedores
=================================================


**Descrição:** Este tutorial tem por finalidade demonstrar como configurar as ferramentas essenciais do Brazil Data Cube, dos quais são:

- `BDC-Catalog <https://github.com/brazil-data-cube/bdc-catalog>`_: Estrutura do modelo de banco de dados relacional para os metadados.
- `BDC-STAC <https://github.com/brazil-data-cube/bdc-stac>`_: Servidor STAC para catalogo de dados do Brazil Data Cube.
- `Cube-Builder <https://github.com/brazil-data-cube/cube-builder>`_: Serviço para geração de cubos de dados de observação da terra.

  - `Data Cube Manager <https://github.com/brazil-data-cube>`_: Interface gráfica para manipulação e geração dos cubos de dados gerados pela ferramenta ``Cube-Builder``.

- `BDC-Collection-Builder <https://github.com/brazil-data-cube>`_: Serviço para coleta, processamento e publicação dos dados de observação da terra. Inclui mecanismos para geração dos produtos `Analysis Ready Data (ARD)`.

Este tutorial é destinado para os usuários que desejam aprender como configurar cada ambiente citado acima e também oferece o foco para usuários que desejam trabalhar em projetos de aplicações relacionadas com as tecnologias do Brazil Data Cube.
O Tutorial consiste nos passos de instalação e uso das ferramentas acima. Em cada seção de serviço será dispobilizado um passo-a-passo de como configurar em modo desenvolvimento e também uma seção descrevendo o deploy em modo produção.
