Open Data Cube: Introdução e Integração com o projeto BDC¶

Descrição: Atualmente o framework Open Data Cube (ODC) e todo seu ecossistema de ferramentas vem sendo testado com os produtos de dados do Brazil Data Cube. O objetivo desta integração é aumentar o conjunto de ferramentas de acesso, visualização e análise disponíveis para os usuários dos diversos cubos de dados produzidos pelo projeto. Este Tech Talk irá apresentar os principais conceitos relativos ao ODC e o status da integração desta tecnologia no projeto BDC. Os participantes serão convidados a realizar um hands-on durante a apresentação.

Apresentadores: Felipe Menino Carlos e Vitor Gomes

Data: 04 de Setembro de 2020

Tópicos:

  • Criando uma Imagem no Docker para o Ambiente ODC
  • Executando os Serviços ODC
  • Configurando o ODC-Core
  • Adicionando um Produto de Dados
  • Extraíndo Metadados dos Datasets
  • Adicionando Datasets ao Banco de Dados do ODC
  • Ingestão de dados
  • Consumindo os dados
